# Java 変数宣言
# 基本構文
## 変数の宣言  
  　型 変数名;   

## 変数初期化  
 　型	変数名=値;  

 ## 型名
|型名|内容|   
|:--:|:--:|   
| boolean | true or false |  
| char   | 16bit Unicode文字 |
| byte   |  8bit 整数 |  
| byte   | 16bit 整数 |
| int    | 32bit 整数 |
| long   | 64bit 整数 |  
| float  | 32bit 単精度浮動小数点数 |  
| double | 64bit 倍精度浮動小数点数 |
