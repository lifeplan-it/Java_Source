# Java 配列変数の宣言

## 基本構文   
###  配列変数の宣言  
    型名[]  配列変数名;  
    配列変数名 = new 型名[要素数] ;  

##  省略型
    型名[] 配列変数名 = new 型名[要素数];  

##  リスト型  
    型名[] 配列変数名 = { 値１, 値２, 値３,・・・} ;
