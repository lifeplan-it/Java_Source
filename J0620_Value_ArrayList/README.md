# Java ArrayList の使い方
 
## ArrayList の宣言
ArrayList<型> 配列変数名 = new ArrayList< 型 > ();

## ArrayList データの追加
    配列変数名.add( 値 );
    配列変数名.add( id, 値 );

## ArrayList データの更新
    配列変数名.set( id, 値 );

## ArrayList データの削除
    配列変数名.remove( id, 値 );
